# 파일 Tool

Claude Code에서 가장 많이 사용되는 Tool로, 모든 파일시스템 읽기/쓰기 작업을 처리한다.

## Read

| 속성 | 값 |
|------|-----|
| 출력 형식 | `cat -n` (1부터 시작하는 줄 번호) |
| 기본 한도 | 파일 시작부터 2,000줄 |
| 지원 | 텍스트, 이미지 (PNG/JPG), PDF (요청당 최대 20페이지; 10페이지 이상은 `pages` 매개변수 **필수**), Jupyter Notebook |

## Write

| 속성 | 값 |
|------|-----|
| 모드 | 전체 파일 덮어쓰기 |
| 전제조건 | 기존 파일이면 먼저 Read 필요 |
| 선호도 | 수정 시에는 Edit 도구 사용 |

## Edit

| 속성 | 값 |
|------|-----|
| 모드 | 정확한 문자열 교체 |
| 전제조건 | 먼저 Read 필요 |
| 유일성 | `old_string`이 파일 내에서 유일해야 함 |
| 배치 모드 | `replace_all: true`로 모든 발생 교체 |

## Glob

| 속성 | 값 |
|------|-----|
| 엔진 | 네이티브 glob 매칭 |
| 패턴 | `**/*.js`, `src/**/*.ts` 등 |
| 정렬 | 수정 시간순 |

## Grep

| 속성 | 값 |
|------|-----|
| 엔진 | ripgrep (`rg`) |
| 패턴 | 전체 정규식 |
| 출력 모드 | `content`, `files_with_matches` (기본), `count` |
| 기본 한도 | 250개 결과 |

## Tool 선택 규칙

| 작업 | 사용 | 사용하지 않음 |
|------|------|------------|
| 파일 읽기 | Read | `cat`, `head`, `tail` |
| 파일 편집 | Edit | `sed`, `awk` |
| 파일 생성 | Write | `echo >`, heredoc |
| 파일 검색 | Glob | `find`, `ls` |
| 콘텐츠 검색 | Grep | `grep`, `rg` |
