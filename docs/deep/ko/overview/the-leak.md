# 유출 경위

## 무슨 일이 있었나

2026년 3월 31일, 보안 연구자 **Chaofan Shou** ([@Fried_rice](https://twitter.com/Fried_rice))가 동부 표준시 오전 4시 23분에 `@anthropic-ai/claude-code` npm 패키지 버전 **2.1.88**에 **59.8MB JavaScript 소스맵 파일**(`.map`)이 포함되어 있음을 발견했다.

이 소스맵은 Anthropic의 Cloudflare R2 버킷에 호스팅된 난독화되지 않은 zip 아카이브를 가리키고 있었으며, 약 **1,900개 파일**에 걸친 **512,000줄의 TypeScript**. Claude Code의 전체 소스코드가 노출되었다.

## 원인

두 가지 요인이 결합된 결과다:

1. **Bun의 기본 동작**: Claude Code는 [Bun](https://bun.sh/)으로 컴파일/번들링되는데, Bun은 번들링 시 기본적으로 소스맵을 생성한다.

2. **`.npmignore` 누락**: `.npmignore` 파일에 생성된 `.map` 파일을 제외하는 항목이 없어, npm 패키지에 그대로 포함되었다.

```
@anthropic-ai/claude-code@2.1.88
├── cli.mjs           (번들된 JavaScript)
├── cli.mjs.map       (59.8 MB 소스맵 ← 유출 원인)
└── package.json
```

## 커뮤니티 반응

유출은 오픈소스 커뮤니티에서 전례 없는 반응을 불러일으켰다:

- 수시간 내에 다수의 분석 레포지토리가 생성됨
- 유출을 문서화한 레포가 **역대 최단 시간 50,000 스타**를 기록 (약 2시간)
- Anthropic은 해당 패키지 버전을 제거하며, 보안 침해가 아닌 인적 오류라고 설명
- 소스코드를 직접 호스팅한 레포에 대해 DMCA 테이크다운 요청이 다수 접수됨

## 밝혀진 것들

유출된 소스에서 이전에 알려지지 않았던 여러 사실이 드러났다:

- **50개 이상의 피처 플래그**: 미출시 기능을 제어하는 플래그
- **KAIROS**: 완전히 구현된 자율 데몬 모드
- **안티 디스틸레이션 메커니즘**: 가짜 도구 주입 포함
- **클라이언트 인증(DRM)**: Zig/Bun HTTP 전송 계층에서 구현
- **내부 모델 코드네임**: Capybara, Fennec, Numbat, Tengu
- **3단계 자가 치유 메모리** 아키텍처
- **다단계 권한 파이프라인**: 자동화된 보안 결정을 위한 AI 분류기
- 110+ 지시사항으로 구성된 완전한 **시스템 프롬프트** 구조

각 발견 사항은 이 문서의 후속 섹션에서 상세히 분석된다.
