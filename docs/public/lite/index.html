The Leak: 유출 경위

2026년 3월 31일, Claude Code npm 패키지에서 59.8MB 소스맵 파일이 발견되어 전체 소스코드가 공개된 사건입니다.

59.8 MB Source Map
~1,900 TS Files
~512K Lines of Code

⚠️
무슨 일이 있었나 @anthropic-ai/claude-code npm 패키지 버전 2.1.88에 59.8MB JavaScript 소스맵 파일(.map)이 포함되어 있었습니다. 이 소스맵은 Cloudflare R2 버킷에 호스팅된 난독화되지 않은 zip 아카이브를 가리키고 있었으며, 약 1,900개 파일에 걸친 512,000줄의 TypeScript 코드가 노출되었습니다.
🔍
원인 두 가지 요인이 결합된 결과였습니다:
  • Bun의 기본 동작 — Bun은 번들링 시 기본적으로 소스맵을 생성합니다.
  • .npmignore 누락 — 생성된 .map 파일을 제외하는 항목이 없어 npm 패키지에 그대로 포함되었습니다.

유출 규모

  • 패키지 버전: @anthropic-ai/claude-code@2.1.88
  • 소스맵 크기: 59.8 MB
  • TypeScript 파일 수: ~1,900개
  • 코드 줄 수: ~512,000줄
  • 발견 시각: 2026년 3월 31일 ET 오전 4:23
  • 50K GitHub 스타 달성 시간: ~2시간 (역대 최단)

44개 피처 플래그 발견

미출시 기능을 제어하는 44개의 피처 플래그가 발견되었습니다. 이 중에는 KAIROS(자율 데몬 모드), 음성 입력, UltraPlan 같은 아직 공개되지 않은 주요 기능들이 포함되어 있습니다.

KAIROS: 자율 데몬 모드

완전히 구현되어 있지만 컴파일 타임 플래그로 비활성화된 자율 에이전트 데몬입니다. <tick> 프롬프트 시스템으로 주기적으로 작업 상태를 평가하고, autoDream으로 유휴 시 메모리를 자동 통합합니다.

안티 디스틸레이션 메커니즘

경쟁사 모델 디스틸레이션을 방지하기 위한 3중 방어:

  • Client Attestation (Zig 레벨 해시)
  • Anti-Distillation (가짜 도구 주입)
  • Reasoning Summarization (서버측 추론 요약)
